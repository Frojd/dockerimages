# postgis-sv_se

PostGIS Docker image with `sv_SE.UTF-8` locale.

Based on the official [postgis/postgis](https://hub.docker.com/r/postgis/postgis) images.

## Supported versions

| Tag      | PostgreSQL | PostGIS |
| -------- | ---------- | ------- |
| `latest` | 17         | 3.5     |
| `17`     | 17         | 3.5     |
| `16`     | 16         | 3.5     |
| `15`     | 15         | 3.5     |
| `14`     | 14         | 3.5     |
| `13`     | 13         | 3.5     |
| `12`     | 12         | 2.5     |
| `11`     | 11         | 2.5     |
| `10`     | 10         | 2.5     |

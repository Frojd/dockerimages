# Docker Images

A collection of Docker images maintained by [Frojd](https://github.com/Frojd).

## WP-CLI

Alpine-based Docker images with [WP-CLI](https://wp-cli.org/) pre-installed. All images include the `mysqli` PHP extension and MySQL client.

Images are published to GitHub Container Registry at `ghcr.io/frojd/dockerimages/`.

### Supported versions

| PHP version | WP-CLI versions |
|---|---|
| 8.1 | 2.8.1, 2.9.0, 2.10.0, 2.11.0, 2.12.0 |
| 8.2 | 2.8.1, 2.9.0, 2.10.0, 2.11.0, 2.12.0 |
| 8.3 | 2.8.1, 2.9.0, 2.10.0, 2.11.0, 2.12.0 |

### Unmaintained versions

| PHP version | WP-CLI versions |
|---|---|
| 7.2 | 2.3.0, 2.4.0 |
| 7.4 | 2.3.0, 2.4.0 |
| 8.0 | 2.7.1 |

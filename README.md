# postgis-sv_se

PostGIS Docker image with `sv_SE.UTF-8` locale.

Based on the official [postgis/postgis](https://hub.docker.com/r/postgis/postgis) images.

## Supported versions

| Tag      | PostgreSQL | PostGIS |
| -------- | ---------- | ------- |
| `latest` | 18         | 3.6     |
| `18`     | 18         | 3.6     |
| `17`     | 17         | 3.5     |
| `16`     | 16         | 3.5     |
| `15`     | 15         | 3.5     |
| `14`     | 14         | 3.5     |

## No longer supported

| PostgreSQL | PostGIS |
| ---------- | ------- |
| 13         | 3.5     |
| 12         | 2.5     |
| 11         | 2.5     |
| 10         | 2.5     |
